# CDC Clock - Eastern Time

A sleek, modern digital clock displaying Eastern Time (ET) with the CDC logo.

## Features

- Real-time clock displaying Eastern Time (automatically adjusts for EST/EDT)
- CDC logo prominently displayed
- Clean, professional design with subtle animations
- Responsive layout for all devices
- No external dependencies (vanilla HTML/CSS/JS)
